4. Upload index.html, recipes.json, and the assets folder.
5. Commit with: Fix approved homepage navigation
6. Wait 1–3 minutes.
7. Open https://jessisouther.github.io/from-my-kitchen/
8. Hard refresh with Ctrl+F5 if the old page remains cached.
